- 一个全英文ASCII文件，client需要发送这个文件
- client需要分片发送该文件
- server需要将收到的文件中的字符反转后返回给client

- 四种报文类型
    - Initialization: type(type=1) | N (N=请求server来reverse的块数)
    - agree: type(type=2)
    - reverseRequest: type(3) | length | Data
    - reverseAnswer: type(4) | length | reverseData


- 流程： client发送init报文，告诉server有多少reverse块 -> server回传agree报文 -> client逐次发送request报文
        -> server do the reverse and return the data ->

- 命令行启动， client需要指定server IP, Port

- 块大小随机， 在[min, max]一个区间里，最后一块除外
    - min, max 也推荐在命令行中指定
    - 需要先分块，计算块数，在进行init报文发送

- client接收到server传回的报文之后，需要打印"第x块：反转的文本"， eg:"4, fldsajlf"

- server能够处理多个client的同时请求，即非阻塞

- socket.setsockopt(), accept()
- str [start:end:step]
